## 🚀 デプロイ方法

### オプション1: Vercel（最も簡単・推奨）

1. **Vercelアカウント作成**
   - https://vercel.com にアクセス
   - GitHubアカウントでサインアップ（無料）

2. **デプロイ**
   - ダウンロードした6つのファイルを1つのフォルダにまとめる
   - Vercelにログイン
   - 「Add New」→「Project」
   - フォルダをドラッグ&ドロップ
   - 「Deploy」ボタンをクリック

3. **数分で完成！**
   - URLが発行されます（例: https://ganbare-note.vercel.app）

### オプション2: Netlify

1. **Netlifyアカウント作成**
   - https://netlify.com にアクセス
   - GitHubアカウントでサインアップ（無料）

2. **デプロイ**
   - フォルダをドラッグ&ドロップ
   - 自動でデプロイ

### オプション3: GitHub Pages

1. **GitHubリポジトリ作成**
   - 新しいpublicリポジトリを作成
   - 6つのファイルをアップロード

2. **GitHub Pages有効化**
   - Settings → Pages
   - Source: main branch
   - 保存

## 📲 iPhoneでの使い方

1. **Safariで開く**
   - デプロイされたURLにアクセス

2. **ホーム画面に追加**
   - 共有ボタン（□↑）をタップ
   - 「ホーム画面に追加」を選択
   - 「追加」をタップ

3. **アプリとして使用**
   - ホーム画面のアイコンから起動
   - アプリのように動作します！

## 🔒 データについて

- すべてのデータはiPhone内（localStorage）に保存
- クラウドには送信されません
- プライバシー安心
- ブラウザのデータを消去すると記録も消えます

- **更新**: ページをリロードすると最新版に更新されます

## ❓ トラブルシューティング

**データが消える**
- ブラウザのプライベートモードを使っていませんか？
- 通常モードで使用してください

**ホーム画面に追加できない**
- Safariで開いていますか？Chrome等では機能しません
- iOS 13以上が必要です

**オフラインで動かない**
- 初回は必ずオンラインでアクセスしてください
- Service Workerがキャッシュを作成します
